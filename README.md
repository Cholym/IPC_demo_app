# IPC_demo_app
It's a study project, which demonstrates IPC messages' stack.

## Links:
https://www.geeksforgeeks.org/ipc-using-message-queues/

## Task:
1. Создайте приложение для записи:
    a.	Создать очередь сообщений
    b.	Напишите 1-е сообщение, которое начинается со строки «writer%STUDENT_ID%» и содержит список всех файлов *.conf, разделенных запятыми, в каталоге /etc.
    c.	Напишите 2-е сообщение, содержащее через запятую список всех каталогов в каталоге /dev.
2.	Создайте приложение для чтения:
    a.	Прочитайте 1-е сообщение из очереди и выведите каждое имя файла на экране в новой строке
    b.	Прочитайте 2-е сообщение, определите время записи последнего сообщения в очередь и выведите его на экран
    c.	Удалите очередь сообщений
